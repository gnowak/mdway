Canvas Guide & Legend

Sticky Notes
Double-Click Body: Edit note text. Click checkmark (or double-click outside) to save.
Drag Border / Corner: Resize note (hold Shift to lock aspect ratio, hold Alt to scale from center).
Paste (Ctrl+V): Drop a new card (pasting a web URL auto-scrapes metadata, pasting an MTG deck list pulls cards automatically!).
Canvas Navigation
Left-Click Drag Background: Pan the infinite board.
Mouse Scroll: Scroll board (customizable: hold Shift to zoom, hold Ctrl to scroll horizontally).
Space / Escape: Reset view to origin (0, 0) at zoom level 1.
